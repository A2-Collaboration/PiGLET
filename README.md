PiGLPlot
========
PiGLPlot is a live graphing tool for EPICS [1] PVs for the
Raspberry Pi. It uses OpenGL for drawing.

[1] http://www.aps.anl.gov/epics/

Dependencies
============

 * cmake
 * libmagickwand-dev
 * GLUT (for PC)
 * /opt/vc headers (for the PI)
 * EPICS (tested with 3.14.12.3 and 3.15.0.1)
